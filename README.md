# goit-rdb-hw-03

Домашнє завдання до Теми 3. Завантаження даних та основи SQL. DQL команди.

## Структура

- `queries.sql` — текст SQL-запитів для всіх 5 завдань.
- `screenshots/` — пронумеровані скриншоти запитів та результатів з MySQL Workbench
  (формат `pN_*.png`, де `N` — номер завдання).

## Опис завдань

1. **Завдання 1.**
   1. Вибрати всі стовпчики з таблиці `products` (через `*`).
   2. Вибрати лише `name`, `phone` з таблиці `shippers`.
2. **Завдання 2.** Знайти `AVG`, `MAX`, `MIN` стовпчика `price` таблиці `products`.
3. **Завдання 3.** Обрати унікальні значення `category_id` та `price` з `products`,
   відсортувати за `price DESC`, обмежити 10 рядками.
4. **Завдання 4.** Кількість продуктів з ціною у межах `20..100` (включно).
5. **Завдання 5.** Кількість продуктів та середня ціна у кожного `supplier_id`.

## Як запускати

1. Імпортувати CSV-датасет з конспекту Теми 3 (таблиці `products`, `shippers` тощо).
2. Відкрити `queries.sql` у MySQL Workbench.
3. Виконати кожен запит окремо та зробити скриншот результату.
